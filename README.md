[dotfiles][projecturl]
======================

Installation
------------

1. Clone repository into a `$HOME/.dotfiles` directory:
   * `git clone git://github.com/AndersDJohnson/dotfiles.git ~/.dotfiles`

2. Create symlinks:
   * For example, `ln -s ~/.dotfiles/screenrc ~/.screenrc`
